Welcome to the IBM w3id SSO Management Utility application, built by IBM Cloud Identity Services.

The functions of this utility are briefly described as follows...

Function Description
Monitor (coming soon)

Monitor (coming soon) the w3id infrastructure at a glance, including:

  • WebSEAL health
  • ISAM health
  • TFIM health
Check / Board Check / Board w3id partner jobs generated by the SSO Self-Service Provisioner.
Download / search logs

Download / search logs from key w3id infrastructure components, including:

  • WebSEAL Request and Message logs
  • ISAM Application Message logs

    • Note: ISAM MGA Message logs fetch have been disabled due to changes in ISAM 9 REST API from ISAM 8; WebSEAL (RP) logs not affected
Admin

Admin functions convenient to supporting key w3id infrastructure components, including:

  • Device fingerprint retrieval
  • F5 cookie decoder
  • ISAM control (Coming soon)
  • TFIM control
    • Custom LDAP Mapper management (Coming soon)
